This repository contains various flowchart implementations created using [Flowgorithm](http://www.flowgorithm.org/), a free beginner's programming language that is based on simple graphical flowcharts. Flowgorithm is an excellent tool for:
- Learning programming logic and concepts
- Creating visual representations of algorithms
- Understanding program flow and control structures
- Transitioning from flowcharts to text-based programming languages

## ğŸ¯ What is Flowgorithm?

Flowgorithm is a graphical authoring tool that allows users to write and execute programs using flowcharts. It helps beginners understand programming concepts by visualizing the logic flow before writing actual code. The tool can also convert flowcharts into multiple programming languages including:
- C#
- C++
- Java
- JavaScript
- Python
- And many more

## ğŸš€ Getting Started

### Prerequisites

To view and run the flowcharts in this repository, you'll need:
- **Flowgorithm** - Download and install from [flowgorithm.org](http://www.flowgorithm.org/download/index.html)
- Operating System: Windows, macOS (via Wine), or Linux (via Wine)

### Installation

1. Download and install Flowgorithm from the official website
2. Clone this repository:
   ```bash
   git clone https://github.com/Siva-Balan-V/Flowgorithms.git
   ```
3. Open Flowgorithm application
4. Navigate to File > Open and select any `.fprg` file from this repository
